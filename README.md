## 🌟 프로젝트 개요
이 프로젝트는 복잡한 **동양 철학 용어(음양, 오행 등)**와 그 기반이 되는 **한자어**를 체계적으로 정리하고 학습하기 위해 기획되었습니다. 단순한 사전 기능을 넘어, Java의 객체지향 특성을 활용한 데이터 관리와 퀴즈 모드를 제공합니다.

## 🚀 주요 기능
1. **용어 사전 (CRUD):** 한자 용어 등록, 상세 설명 조회, 수정 및 삭제.
2. **학습 데이터 영속화:** 파일 입출력(File I/O)을 통한 데이터 저장 및 불러오기.
3. **학습 모드:** 무작위 단어 추출을 통한 한자 퀴즈 기능.
4. **GUI 인터페이스:** Swing을 활용한 사용자 친화적 화면 (구현 예정).

## 📂 프로젝트 구조
- `src/`: 자바 소스 코드
- `data/`: 용어 데이터 파일 (.txt / .csv)
- `docs/`: 기획서 및 설계 문서
